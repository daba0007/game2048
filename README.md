## 2048游戏
我使用js编写了一个2048的运行环境，只要在界面上传一个ai.js文件,文件中存在函数
```js
function AI(score,board){
  var nextmove = "Left"   // Left,Right,Up,Down
  return nextmove;
}
```
就可以实现ai的行走

AI的行走策略是通过棋盘决定的，通过传入棋盘来产生策略，并返回下一步，即可实现ai行走2048.
